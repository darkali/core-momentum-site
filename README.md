# Core Momentum ⚙️

Core Momentum is a fictional wellness and productivity-focused brand designed to demonstrate a modern and bold layout for a small business or consultancy website. This static site showcases a clean, functional design ideal for performance coaching, tech wellness brands, or lifestyle services.

## 🚀 Live Preview

👉 [View the Site](https://darkali.github.io/core-momentum-site/)  

---

## 📌 Features

- Multi-page structure: Home, Programs, Blog, About, Contact
- Clean, bold visual style for a confident brand presence
- Semantic HTML5 and modular CSS
- Highlighted services and call-to-action sections
- Contact form for inquiries
- Responsive design structure
